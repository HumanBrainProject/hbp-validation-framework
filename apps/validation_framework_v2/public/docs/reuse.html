Simulate

Run simulations with single-cell models

For some models of single neurons, you will see this button:

This opens the neuron model in the BlueNaaS app ("Neuron as a Service"), which allows you to run simulations with the NEURON simulator from within your web browser.

Simulation tools for other types of models, including network models for the NEST simulator and models for neuromorphic hardware, will be added over the next months, so check back regularly!
